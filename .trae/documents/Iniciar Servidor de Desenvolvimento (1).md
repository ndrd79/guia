## Objetivo
- Iniciar o servidor Next.js em desenvolvimento para visualizar as alterações na página inicial e admin.

## Passos
- Executar `npm run dev` no diretório do projeto.
- Aguardar a inicialização e confirmar a URL local (`http://localhost:3000`).

## Verificação
- Abrir `http://localhost:3000/` para validar carregamento básico.
- Se necessário, acessar `http://localhost:3000/admin/login` e `http://localhost:3000/admin/banners` para validar o painel.
